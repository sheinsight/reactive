# Installation \{#installation}

You can install Reactive from the [npm](https://npm.im/@shined/reactive) registry using your package manager of choice.

import { PackageManagerTabs } from 'rspress/theme'

<PackageManagerTabs
  command={{
    npm: 'npm install @shined/reactive',
    yarn: 'yarn add @shined/reactive',
    pnpm: 'pnpm add @shined/reactive',
    bun: 'bun add @shined/reactive',
  }}
/>
